# 🍗 Telegram Food Ordering Bot

A fully functional **Telegram-based Food Ordering Bot** built using **Python** and **python-telegram-bot v20**.  
The bot allows customers to browse a menu, place food orders, choose payment methods, and notifies the owner instantly.

🚀 **Live & Deployed on Render (24×7)**

---

## ✨ Features

- 📋 Interactive food menu
- 🛒 Step-by-step order flow
- 🔢 Quantity selection
- ➕ Add more items option
- 👤 Customer details (Name, Phone, Address)
- 💳 Multiple payment options:
  - UPI
  - QR Code
  - Cash on Delivery
- 🔔 Instant owner notification for every order
- 🔐 Secure token handling using environment variables
- ☁️ Deployed on Render (free tier)

---

## 🧠 Tech Stack

- **Language:** Python 3.11
- **Telegram Library:** python-telegram-bot v20+
- **Hosting:** Render
- **Architecture:** Long Polling (No Webhook)
- **Payments:** UPI deep link + QR image
